# groovy-cli-gradle-hello-world

## Description
A POC for a groovy gradle build.

## Tech stack
- groovy

## Docker stack
- gradle:4.2.1-jdk8-alpine

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
